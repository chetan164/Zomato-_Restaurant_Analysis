# Zomato-_Restaurant_Analysis 
# Project Overview
This project focuses on analyzing Zomato restaurant data to gain insights into customer preferences, restaurant performance, pricing, and service availability. The analysis helps understand food trends, popular locations, rating patterns, and cost distribution across different cities and cuisines.

# Dataset used -> [Zomato Dataset](https://drive.google.com/your-link-here)
- Zomato restaurant listing data  
- Each row represents one restaurant  
- Contains location, cuisine, rating, and pricing details  
- Includes online delivery and table booking information  
- Useful for analyzing food trends and customer preferences  

# Objectives
- Analyze restaurant ratings and customer feedback
- Identify top cities and locations with the highest number of restaurants
- Study pricing trends and affordability
- Compare online delivery and table booking services
- Analyze popular cuisines and restaurant distribution

# Tools & Technologies Used
- **Power BI** – Data visualization and dashboard creation  
- **Power Query** – Data cleaning and transformation  
- **DAX** – Measures and calculated columns  
- **Dataset** – Zomato restaurant dataset  

## Dashboard & Analysis
- The Power BI dashboard includes:
- Restaurant count by city and country  
- Average rating analysis  
- Price range vs rating comparison  
- Online delivery & table booking availability  
- Cuisine-wise restaurant distribution  


## Key Insights
- Major cities like **Delhi NCR, Bangalore, and Mumbai** have the highest restaurant density  
- Restaurants offering **online delivery** generally receive higher ratings  
- Most restaurants fall into **low to mid price ranges**, showing affordability  
- **North Indian and Chinese cuisines** are the most popular  
- Higher-rated restaurants often provide **table booking facilities**

## Conclusion
This project demonstrates how **Power BI** can convert raw data into valuable insights.  
It helps understand market trends, customer preferences, and restaurant performance, enabling **data-driven decision making**.
